Retención cero — por diseño Zero retention — by design

  • Los PDFs nunca se escriben en disco. Se procesan en memoria y se descartan inmediatamente. PDFs are never written to disk. Processed in memory and discarded immediately.
  • El informe aparece en pantalla. Cuando cierras la pestaña, desaparece. No guardamos nada. The report appears on screen. When you close the tab, it's gone. We keep nothing.
  • Sólo registramos: marca de tiempo, tamaño del archivo, código utilizado. Sin contenido, sin nombres. We only log: timestamp, file size, code used. No content, no names.
  • La API de Anthropic no entrena modelos con datos enviados. Política de datos → Anthropic's API does not train on submitted data. Data policy →
  • Código abierto y auditable. Ver en GitHub → Open-source and auditable. View on GitHub →
