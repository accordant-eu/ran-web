    Retención cero — por diseño / Zero retention — by design

    • Los PDFs nunca se escriben en disco. Se procesan en memoria (BytesIO) y se descartan al final de la petición. PDFs are never written to disk. Processed in memory and discarded.
    • El informe se muestra en pantalla y nunca se almacena. The report is shown on screen and never stored.
    • El único registro operacional: marca de tiempo, tamaño del archivo, código utilizado. Sin contenido, sin nombres. The only operational record: timestamp, file size, code used. No content, no names.
    • La API de Anthropic no usa los datos enviados para entrenar modelos. Política de datos de Anthropic → Anthropic's API does not use submitted data for model training. Anthropic data policy →
    • El código es abierto y auditable. github.com/accordant-eu/ran-web → The code is open-source and auditable.
